# Project Context

This repository is a deliberately narrow two-minute demo for a Salesforce GTM Product Marketing Manager portfolio.

## Product promise
Monitor selected competitor webpages, compare weekly snapshots, and generate a concise PMM brief explaining:
1. What changed
2. Why it matters
3. What the GTM team should do

## Product constraints
- Do not turn this into a full operating system or broad competitive-intelligence platform.
- Preserve the three-step story: Observation → Inference → Recommended Action.
- Favor a clean, credible demo over architectural complexity.
- Live AI analysis must use the locally authenticated Claude Code CLI (`claude -p`), not a hard-coded API key.
- Never commit credentials or personal data.
- Keep the app runnable on macOS with Python and Streamlit.

## Definition of done
- Demo mode works immediately.
- Live mode can capture two snapshots and produce a change brief.
- Errors are explained clearly.
- README allows a recruiter to understand the value in under 30 seconds.
